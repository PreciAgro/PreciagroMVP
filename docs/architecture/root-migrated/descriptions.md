# Preciagro Engine Descriptions

## Core Intelligence Engines

### crop_intelligence
Crop growth reasoning and agronomic logic.

### image_analysis
Vision engine for diseases, pests, weeds and nutrient issues.

### diagnosis_recommendation
Generates actionable farmer guidance.

### geocontext
Location, climate and soil contextualization.

### temporal_logic
Seasonality and time-based scheduling.

## Language & Communication Engines

### agro_llm
Core agricultural language model.

### conversational_nlp
Chat and intent understanding engine.

## Data & Integration Engines

### data_integration
Weather, satellite and external data ingest.

### model_orchestration
Handles model routing and pipelines.

### data_governance_lineage
Dataset and model lineage tracking.

## Infrastructure & Access Engines

### api_gateway
Routed entry point for apps.

### security_access
Auth, encryption and key access.

## User Experience Engines

### ux_orchestration
Coordinates UI results and flows.

### trust_explainability
Provides model reasoning explanations.

## Farmer & Farm Management Engines

### farmer_profile
Stores farmer history and preferences.

### farm_inventory
Tracks farm input stock.

## Learning & Evaluation Engines

### feedback_learning
Collects feedback for continuous learning.

### evaluation_benchmarking
Evaluates model performance.

## Product Insights

### pie_lite
Early product insights engine.
